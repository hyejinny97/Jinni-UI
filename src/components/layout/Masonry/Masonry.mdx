# Masonry

2차원 구조에서 다양한 items가 각각 동일한 너비와 서로 다른 높이를 가진 형태로 나열되도록 함

- items는 row 방향으로 적재됨
- 가장 짧은 column에 우선적으로 적재됨

## Import

```js
import { Masonry } from '@/components/layout/Masonry';
```
