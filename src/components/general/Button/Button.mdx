# Button

form을 제출하거나, 모달창을 여는 등 action이나 event를 유발/취소/삭제하기 위해서 사용됨

## Import

```js
import { Button } from '@/components/general/Button';
```

## Usage

- `children`: 버튼 내부 내용(label)

### Button adornments

- `startAdornment`: content 앞에 위치하는 부가 요소 (icon, avatar 등)
- `endAdornment`: content 뒤에 위치하는 부가 요소 (icon, avatar 등)
