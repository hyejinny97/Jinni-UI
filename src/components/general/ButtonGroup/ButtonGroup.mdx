# ButtonGroup

연관된 버튼들의 모음

## Import

```js
import { ButtonGroup } from '@/components/general/ButtonGroup';
```

## Usage

### Basic ButtonGroup

- `children`: `Button` 컴포넌트들

<Canvas of={ButtonGroupStories.BasicButtonGroup} />

### Variants

- `variant`: 버튼 종류

<Canvas of={ButtonGroupStories.Variants} />

### Sizes

- `size`: 버튼 크기

<Canvas of={ButtonGroupStories.Sizes} />

### Color

- `color`: 버튼 색상(theme/palette 포함)

<Canvas of={ButtonGroupStories.Color} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={ButtonGroupStories.Disabled} />

### Vertical Button Group

- `orientation`: 버튼 방향

<Canvas of={ButtonGroupStories.VerticalButtonGroup} />

### Overlay Effect

- overlay effect: (disabled가 아닌 경우) hover 시 반투명의 overlay가 생성됨

#### 1) overlay color

- `overlayColor`: overlay 색상
  - default: variant === 'filled'면 'white', 아니면 'black'

<Canvas of={ButtonGroupStories.OverlayColor} />

#### 2) disable overlay

- `disableOverlay`: true이면, overlay가 나타나지 않음

<Canvas of={ButtonGroupStories.DisableOverlay} />

### Ripple Effect

- ripple effect: (disabled가 아닌 경우) 클릭 시 ripple 애니메이션이 생성됨

#### 1) ripple color

- `rippleColor`: ripple 색상
  - default: variant === 'filled'이면 'white', 아니면 'black'

<Canvas of={ButtonGroupStories.RippleColor} />

#### 2) ripple start location

- `rippleStartLocation`: ripple 시작점

<Canvas of={ButtonGroupStories.RippleStartLocation} />

#### 3) disable ripple

- `disableRipple`: true이면, ripple effect가 비활성화됨

<Canvas of={ButtonGroupStories.DisableRipple} />

### Elevation Effect

- elevation effect: (elevation prop을 설정한 경우) hover와 클릭 시 lift 효과가 나타남
- `elevation`: elevation(box-shadow) 정도

<Canvas of={ButtonGroupStories.Elevation} />

<Canvas of={ButtonGroupStories.SplitButton} />

## Props

- `Button` 컴포넌트의 일부 props를 상속받음

  - variant
  - elevation
  - size
  - color
  - disabled
  - overlayColor
  - disableOverlay
  - rippleColor
  - rippleStartLocation
  - disableRipple

- `ButtonGroup` 컴포넌트에 적용된 위 `Button` props는 전부 하위 `Button` 컴포넌트에 일괄 적용됨
