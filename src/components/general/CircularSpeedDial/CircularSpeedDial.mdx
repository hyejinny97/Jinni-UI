# CircularSpeedDial

여러 개의 floating action buttons를 원형으로 띄워줌

- 구성 컴포넌트
  - `CircularSpeedDial` 컴포넌트: 하위 `CircularSpeedDialAction`컴포넌트를 감싸는 wrapper 컴포넌트
  - `CircularSpeedDialAction` 컴포넌트: `Button` + `Tooltip`

## Import

```js
import {
  CircularSpeedDial,
  CircularSpeedDialAction
} from '@/components/general/CircularSpeedDial';
```

## Usage

### Basic CircularSpeedDial

- `CircularSpeedDial` 컴포넌트

  - `children`: action buttons (ex) `CircularSpeedDialAction` 컴포넌트 등)
  - `anchorElRef`: anchor element의 reference
  - `open`: true이면, action buttons가 나타남
  - `onClose`: Escape 키/Background 클릭/MouseLeave/Blur 이벤트가 발생 시 호출되는 함수

- `CircularSpeedDialAction` 컴포넌트

  - `tooltipProps`: tooltip 컴포넌트의 props

### Circular Speed Dial with FAB

- anchor가 Floating Action Button인 경우, `CircularSpeedDial`는 viewport 위에 고정되어야 함
- `CircularSpeedDial` 컴포넌트
  - 내부적으로 `Popper` 컴포넌트를 렌더함
  - `positionType`: popper의 position type
