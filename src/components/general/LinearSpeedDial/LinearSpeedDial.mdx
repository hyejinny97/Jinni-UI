# LinearSpeedDial

여러 개의 floating action buttons를 선형으로 띄워줌

## Import

```js
import {
  LinearSpeedDial,
  LinearSpeedDialAction
} from '@/components/general/LinearSpeedDial';
```

## Usage

### Basic LinearSpeedDial

- `LinearSpeedDial` 컴포넌트

  - `children`: open 됐을 때 나타나는 floating action buttons (ex) `LinearSpeedDialAction` 컴포넌트들)
  - `anchorElRef`: anchor element의 reference
  - `open`: true이면, FABs가 나타남
  - `onOpen`: open을 요청할 때 호출되는 함수
  - `onClose`: close를 요청할 때 호출되는 함수

- `LinearSpeedDialAction` 컴포넌트

  - `tooltipProps`: tooltip 컴포넌트의 props
  - `Button` 컴포넌트의 props를 전부 상속 받음
