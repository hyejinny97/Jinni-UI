# LinearSpeedDial

여러 개의 floating action buttons를 선형으로 띄워줌

- 구성 컴포넌트
  - `LinearSpeedDial` 컴포넌트: 하위 `LinearSpeedDialAction`컴포넌트를 감싸는 wrapper 컴포넌트
  - `LinearSpeedDialAction` 컴포넌트: `Button` + `Tooltip`

## Import

```js
import {
  LinearSpeedDial,
  LinearSpeedDialAction
} from '@/components/general/LinearSpeedDial';
```

## Usage

### Basic LinearSpeedDial

- `LinearSpeedDial` 컴포넌트

  - `children`: action buttons (ex) `LinearSpeedDialAction` 컴포넌트 등)
  - `anchorElRef`: anchor element의 reference
  - `open`: true이면, action buttons가 나타남
  - `onClose`: Escape 키/Background 클릭/MouseLeave/Blur 이벤트가 발생 시 호출되는 함수

- `LinearSpeedDialAction` 컴포넌트

  - `tooltipProps`: tooltip 컴포넌트의 props

<Canvas of={LinearSpeedDialStories.BasicLinearSpeedDial} />

### Linear Speed Dial with FAB

- anchor가 Floating Action Button인 경우, `LinearSpeedDial`는 viewport 위에 고정되어야 함
- `LinearSpeedDial` 컴포넌트
  - 내부적으로 `Popper` 컴포넌트를 렌더함
  - `positionType`: popper의 position type
