# Text

Jinni typography, color 적용이 가능한 글자

## Import

```js
import { Text } from '@/components/general/Text';
```

### Basic Text

- `children`: 텍스트

<Canvas of={TextStories.BasicText} />

### HTML Semantic Element

- `className`을 통해 타이포그래피 적용 가능
