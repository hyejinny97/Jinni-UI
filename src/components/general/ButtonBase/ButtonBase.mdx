# ButtonBase

ripple effect가 있고 기본적인 html button 요소의 특징을 가짐

## Import

```js
import { ButtonBase } from '@/components/general/ButtonBase';
```

## Usage

### Basic ButtonBase

- `children`: 버튼 내부 내용

<Canvas of={ButtonBaseStories.BasicButtonBase} />

### Link Button

- `href`: 링크 url (이 prop이 설정되면 기존 button 태그에서 a 태그로 변환됨)

<Canvas of={ButtonBaseStories.LinkButton} />

### Overlay Effect

- overlay effect: (disabled가 아닌 경우) hover 시 반투명의 overlay가 생성됨

#### 1. Overlay color

- `overlayColor`: overlay 색상

<Canvas of={ButtonBaseStories.OverlayColor} />

#### 2. Disable overlay

- `disableOverlay`: true이면, overlay가 나타나지 않음

<Canvas of={ButtonBaseStories.DisableOverlay} />

### Ripple Effect

- ripple effect: (disabled가 아닌 경우) 클릭 시 ripple 애니메이션이 생성됨

#### 1. Ripple color

- `rippleColor`: ripple 색상

<Canvas of={ButtonBaseStories.RippleColor} />

#### 2. Disable ripple

- `disableRipple`: true이면, ripple effect가 비활성화됨

<Canvas of={ButtonBaseStories.DisableRipple} />

#### 3. Ripple start location

- `rippleStartLocation`: ripple 시작점
  - 'center': 버튼 중앙에서 ripple이 시작됨
  - 'clicked': 버튼 내 클릭한 위치에서 ripple이 시작됨

<Canvas of={ButtonBaseStories.RippleStartLocation} />

### Elevation Effect

- elevation effect: (elevation prop을 설정한 경우) hover와 클릭 시 lift 효과가 나타남
- `elevation`: elevation(box-shadow) 정도

<Canvas of={ButtonBaseStories.Elevation} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={ButtonBaseStories.Disabled} />

<Canvas of={ButtonBaseStories.ComplexButton} />
