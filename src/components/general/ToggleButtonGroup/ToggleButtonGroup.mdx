# ToggleButtonGroup

연관된 option button들을 그룹핑함

## Import

```js
import { ToggleButtonGroup } from '@/components/general/ToggleButtonGroup';
```

## Usage

### Basic ToggleButtonGroup

- Selection 타입

  1. **Exclusive Selection**: 1개 이하의 toggle button만 선택 가능
  2. **Multiple Selection**: 0개 이상의 toggle button 선택 가능

- 기본적으로 Exclusive Selection 상태임

<Canvas of={ToggleButtonGroupStories.BasicToggleButtonGroup} />

### Default value

- `defaultValue`: 초기 selected value (uncontrolled 상태일 때만 적용 가능)

#### 1) Exclusive Selection

- `defaultValue` 타입이 `string` | `number` | `boolean` | `null`인 경우

<Canvas of={ToggleButtonGroupStories.ExclusiveDefaultValue} />

#### 2) Multiple Selection

- `defaultValue` 타입이 `Array<string | number | boolean>`인 경우

<Canvas of={ToggleButtonGroupStories.MultipleDefaultValue} />

### Controlled ToggleButtonGroup

- `value` prop을 통해 ToggleButtonGroup을 control 가능
- `value`: selected value
- `onChange`: selected value가 변경됐을 때 호출되는 함수

#### 1) Exclusive Selection

- `value` 타입이 `string` | `number` | `boolean` | `null` 인 경우

<Canvas of={ToggleButtonGroupStories.ExclusiveControlled} />

#### 2) Multiple Selection

- `value` 타입이 `Array<string | number | boolean>`인 경우

<Canvas of={ToggleButtonGroupStories.MultipleControlled} />
