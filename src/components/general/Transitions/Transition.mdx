# Transitions

Transitions은 UI를 좀 더 풍부하고 사용하기 쉽게 만들어 줌<br />
(아래 Collapse, Fade, Slide, Scale 컴포넌트는 Transition 컴포넌트를 기반으로 만들어짐)

## Import

```js
import {
  Transition,
  Collapse,
  Fade,
  Slide,
  Scale
} from '@/components/general/Transitions';
```

## Usage

### Basic Transition

- `in`: true이면, transition in 됨

- `easing`: transition timing function

  - type: `EasingType` | `string` | `{ enter?: EasingType | string, exit?: EasingType | string }`
    - `EasingType`:
      - `'emphasized'` | `'emphasized-accelerate'` | `'emphasized-decelerate'`
      - `'standard'` | `'standard-accelerate'` | `'standard-decelerate'`
    - `string`:
      - `‘ease’` | `‘ease-in’` | `‘ease-out’` | `‘ease-in-out’`
      - `‘linear’` | `‘step-start’` | `‘step-end’`
      - `‘step(n, <jump-term>)’`
      - `‘cubic-bezier(p1, p2, p3, p4)’`

- `duration`: transition duration (단위: ms)
  - type: `DurationType` | `number` | `{ enter?: DurationType | number, exit?: DurationType | number }`
    - `DurationType`:
      - `'short1'` | `'short2'` | `'short3'` | `'short4'`
      - `'medium1'` | `'medium2'` | `'medium3'` | `'medium4'`
      - `'long1'` | `'long2'` | `'long3'` | `'long4'`
      - `'extra-long1'` | `'extra-long2'` | `'extra-long3'` | `'extra-long4'`

### Slide

#### 1) Default Slide Transition

- 화면의 edge에서부터 slide in 되고, 해당 edge로 slide out 됨

<Canvas of={SlideStories.DefaultSlideTransition} />

#### 2) Direction

- `direction`: transition 방향
  - 'down': 위 → 아래로 내려옴
  - 'up': 아래 → 위로 올라옴
  - 'left': 오른쪽 → 왼쪽으로 이동함
  - 'right': 왼쪽 → 오른쪽으로 이동함

<Canvas of={SlideStories.Direction} />

#### 3) Slide Container

- `container`: slide transition이 나타나는 container
