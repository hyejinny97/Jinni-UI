# Tabs

다른 view로 전환하기 쉽게 도와줌

## Import

```js
import { Tabs } from '@/components/navigation/Tabs';
```

- `value`: selected tab value
- `onChange`: selected tab value가 변경됐을 때 호출되는 함수

### Full Width

- `fullWidth`: true이면, 허용 가능한 너비에 동일한 비율로 꽉 채워짐

<Canvas of={TabsStories.FullWidth} />

### Scrollable

#### 1) scrollable

- `scrollable`: true이면, 수평(또는 수직) scrolling(swiping)이 가능함

<Canvas of={TabsStories.Scrollable} />

#### 2) ScrollButtons

- `scrollButtons`: scrollable가 true일 때, scroll buttons이 나타나는 방식
  - `'auto'`: 뷰포트가 desktop(1200px) 크기 이상일 때만 나타나고, 미만이면 나타나지 않음
  - `false`: 뷰포트 크기에 상관없이 항상 나타나지 않음
  - `true`: 뷰포트 크기에 상관없이 항상 나타남
