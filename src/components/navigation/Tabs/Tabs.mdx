# Tabs

다른 view로 전환하기 쉽게 도와줌

## Import

```js
import {
  Tabs,
  TabList,
  Tab,
  TabPanel,
  TabPrevButton,
  TabNextButton
} from '@/components/navigation/Tabs';
```

## Usage

### Basic Tabs

- `Tabs` 컴포넌트
  - `defaultValue`: 초기 selected tab value
- `Tab` 컴포넌틑
  - `value`: 다른 tab과 구별되는 유일한 값
- `TabPanel` 컴포넌트
  - `value`: 다른 tab panel과 구별되는 유일한 값

<Canvas of={TabsStories.BasicTabs} />

### Controlled Tabs

- `Tabs` 컴포넌트
  - `value`: selected tab value
  - `onChange`: selected tab value가 변경됐을 때 호출되는 함수

<Canvas of={TabsStories.ControlledTabs} />

- `TabList` 컴포넌트

- `Tabs` 컴포넌트

### Color

- `TabList` 컴포넌트

### Full Width

- `TabList` 컴포넌트
  - `fullWidth`: true이면, 허용 가능한 너비에 동일한 비율로 꽉 채워짐

- `TabList` 컴포넌트를 통해 하위 모든 `Tab` 컴포넌트에 `Button` 컴포넌트의 일부 props를 일괄 적용 가능

### Scrollable

#### 1. Basic

- `TabList` 컴포넌트
  - `scrollable`: true이면, 수평(또는 수직) scrolling(swiping)이 가능함

<Canvas of={TabsStories.Scrollable} />

#### 2. Scroll buttons

- `TabPrevButton`와 `TabNextButton` 컴포넌트를 사용해 tab 하나씩 이동 가능함
- `TabPrevButton`, `TabNextButton` 컴포넌트는 내부적으로 `Button` 컴포넌트를 렌더하고 있기 때문에 ButtonProps 적용 가능
