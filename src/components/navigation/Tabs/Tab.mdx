# Tab

tabs의 한 부분

## Import

```js
import { Tab } from '@/components/navigation/Tabs';
```

## Usage

### Basic Tab

- `children`: tab label
- `value`: 다른 tab과 구별되는 특정 tab value
  - 값을 직접 임명하지 않는다면, tabs 내에서 이 tab의 position index(number)가 value로 자동 할당됨
