# Menu

특정 이벤트(클릭 등)에 의해 일시적으로 나타나는 선택 가능한 리스트

## Import

```js
import { Menu } from '@/components/navigation/Menu';
```

## Usage

### Basic Menu

- `Menu`, `MenuList`, `MenuItem` 컴포넌트

  - 내부적으로 `Menu` 컴포넌트 안에 `MenuList` 컴포넌트가 `children`을 감싸고 있음
  - `Menu`의 `children`으로 `MenuItem`, `Divider` 등을 채워넣을 수 있음

- `Menu` 컴포넌트

  - `anchorElRef`: anchor가 되는 HTML element의 reference로, 메뉴의 위치를 결정해줌
  - `open`: true이면, 메뉴가 나타남
  - `onClose`: 아래와 같은 이벤트 발생 시 호출되는 함수
    - reason: `'escapeKeydown'`, `'backdropClick'`, `'tabKeyDown'`

- `MenuList` 컴포넌트
  - 아래/위 방향키 클릭 시, 위/아래 방향으로 메뉴 아이템이 포커스됨
  - 사용자가 누른 알파벳 키가 메뉴 아이템의 첫 글자와 같은 경우 포커스됨

### Menu Position

- `Menu` 컴포넌트
  - `menuOrigin`: anchor의 origin에 부착할 menu의 origin
  - `anchorReference`: 메뉴의 위치를 설정할 때, 어떤 anchor prop을 참조할지 결정
    - 'anchorEl'(default) | 'anchorPosition'
  - `anchorOrigin`: menu의 origin이 부착될 anchorEl의 origin
  - `anchorPosition`: client area에서 메뉴의 상대적인 위치
