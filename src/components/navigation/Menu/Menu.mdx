# Menu

특정 이벤트(클릭 등)에 의해 일시적으로 나타나는 선택 가능한 리스트

- `Menu` 컴포넌트
  - anchor에 이벤트가 발생했을 때 적절한 위치에 `MenuList`를 띄우는 컴포넌트
- `MenuList` 컴포넌트
  - `MenuItem`을 담는 컴포넌트
- `MenuItem` 컴포넌트
  - 클릭 가능한 메뉴 항목
- 키보드로 조작 가능
  - 방향키(`ArrowUp`, `ArrowDown`)로 항목 간 이동
  - `Escape`/ `Tab` 키로 메뉴 닫기
  - `Enter` / `Space`로 항목 선택
  - 사용자가 누른 알파벳 키가 메뉴 아이템의 첫 글자와 일치하는 경우 포커스됨

## Import

```js
import { Menu } from '@/components/navigation/Menu';
```

## Usage

### Basic Menu

- `children`: MenuList 컴포넌트 안에 담을 요소 (MenuItem, Divider 등)
- `anchorElRef`: anchor 요소 참조 객체
- `open`: true이면, 메뉴가 나타남
- `onClose`: Escape 키/Tab 키/backdrop 클릭 이벤트가 발생 시 호출되는 함수

### Menu Position

- `menuOrigin`: anchor의 origin에 부착할 menu의 origin
- `anchorReference`: 메뉴의 위치를 설정할 때, 어떤 anchor prop을 참조할지 결정
- `anchorOrigin`: menu의 origin이 부착될 anchorEl의 origin
- `anchorPosition`: client area에서 메뉴의 상대적인 위치
