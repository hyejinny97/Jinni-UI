# Link

Jinni typography, color(theme/palette) 적용이 가능한 anchor element

## Import

```js
import { Link } from '@/components/navigation/Link';
```

## Usage

### Basic Link

- `children`: 링크 콘텐츠
- `as` prop을 통해서 다른 컴포넌트(ex) Routing Library/Next.js의 `Link`)로 변경 가능

<Canvas of={LinkStories.BasicLink} />

### Typography

- `className`을 통해 텍스트에 타이포그래피(15종류) 적용 가능

<Canvas of={LinkStories.Typography} />

### Color

- `style` prop을 통해 color theme/palette 접근 가능

<Canvas of={LinkStories.Color} />

### Line Clamp

- `lineClamp`: 보여지는 라인 수
  - 라인 수가 lineClamp 이상인 경우, 말줄임 표시됨

<Canvas of={LinkStories.LineClamp} />

### Underline

- `underline`: 링크 underline

<Canvas of={LinkStories.Underline} />
