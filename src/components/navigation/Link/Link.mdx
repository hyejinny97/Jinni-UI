# Link

Jinni typography, color 적용이 가능한 anchor element

## Import

```js
import { Link } from '@/components/navigation/Link';
```

## Usage

### Basic Link

- `children`: 링크 콘텐츠
- `as` prop을 통해서 다른 컴포넌트(ex) Routing Library/Next.js의 `Link`)로 변경 가능
