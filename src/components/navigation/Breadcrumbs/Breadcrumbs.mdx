# Breadcrumbs

사이트의 계층적 구조에 따라 페이지를 시각화한 링크의 나열

## Import

```js
import { Breadcrumbs } from '@/components/navigation/Breadcrumbs';
```

## Usage

### Collapsed Breadcrumbs

#### 1. Max items

- `maxItems`: 화면 상에 나타나는 최대 Breadcrumbs 갯수
- ellipse 버튼 클릭 시, 생략된 아이템들이 전부 나타남

<Canvas of={BreadcrumbsStories.MaxItems} />

#### 2. Items before collapse

- `itemsBeforeCollapse`: Breadcrumbs 개수가 maxItems 값 이상일 때, ellipse 버튼 이전에 나타나는 아이템 개수

<Canvas of={BreadcrumbsStories.ItemsBeforeCollapse} />

#### 3. Items after collapse

- `itemsAfterCollapse`: Breadcrumbs 개수가 maxItems 값 이상일 때, ellipse 버튼 이후에 나타나는 아이템 개수

<Canvas of={BreadcrumbsStories.ItemsAfterCollapse} />
