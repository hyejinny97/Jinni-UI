# Dots

dot의 모음으로 pagination을 나타냄

## Import

```js
import { Dots, Dot } from '@/components/navigation/Dots';
```

## Usage

### Basic Dots

- `children`: `Dot` 컴포넌트들
- `defaultValue`: 초기 active dot의 value (uncontrolled 상태일 때만 적용 가능)

<Canvas of={DotsStories.BasicDots} />

### Controlled Dots

- `value` prop을 통해 `Dots`를 control 가능
- `value`: active dot의 value
- `onChange`: active dot value가 변경됐을 때 호출되는 함수

<Canvas of={DotsStories.ControlledDots} />

### Max Dots

- `max`: 화면 상에 나타나는 최대 `Dot` 갯수

<Canvas of={DotsStories.MaxDots} />

### Orientation

- `orientation`: dots 방향

<Canvas of={DotsStories.Orientation} />

### Dot props

- `Dots` 컴포넌트에 적용된 일부 Dot props는 하위 모든 `Dot`에 일괄 적용됨
- `Dot` 컴포넌트는 내부적으로 `ButtonBase` 컴포넌트를 반환하므로, `ButtonBase`의 일부 props 사용 가능
