# Drawer

현 페이지를 벗어나지 않고 손쉽게 페이지 이동이나 기능 처리를 가능하게 함

## Import

```js
import {
  Drawer,
  DrawerHeader,
  DrawerBody,
  DrawerFooter
} from '@/components/navigation/Drawer';
```

## Usage

### Basic Drawer

- `children`: drawer 콘텐츠 (`DrawerHeader`, `DrawerBody`, `DrawerFooter` 등)
- `open`: true이면, drawer가 나타남
- `onClose`: Escape 키/backdrop 클릭 이벤트가 일어날 때 호출되는 함수

<Canvas of={DrawerStories.BasicDrawer} />

### Placement

- `anchorOrigin`: drawer가 위치할 anchor의 origin

<Canvas of={DrawerStories.Placement} />

### Variant

- `variant`: drawer 종류
- `container`: portal container

#### 1. Temporary

- 다른 content보다 상위 level에 drawer가 띄워짐
- open/close 토글 가능
- `onClose` 함수 호출됨

<Canvas of={DrawerStories.Temporary} />

#### 2. Permanent

- 다른 content와 동일 level에 drawer가 띄워짐
- 항상 open된 상태
- `onClose` 함수 호출하지 않음
