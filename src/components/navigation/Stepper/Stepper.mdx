# Stepper

일정 개수의 steps의 progress를 나타냄

## Import

```js
import {
  Stepper,
  Step,
  StepDot,
  StepConnector,
  StepContent,
  StepTitle,
  StepDescription
} from '@/components/navigation/Stepper';
```

## Usage

### Basic Stepper

- `Stepper` 컴포넌트

  - `children`: `Step` 컴포넌트들

- `Step` 컴포넌트

  - `children`: `StepDot`, `StepContent`, `StepConnector` 컴포넌트
  - `status`: step 상태

- `StepDot` 컴포넌트

  - `children`: dot 내부 내용

- `StepContent` 컴포넌트

  - `children`: `StepTitle`, `StepDescription` 컴포넌트

- `StepTitle` 컴포넌트

  - `children`: step title

- `StepDescription` 컴포넌트

  - `children`: step description

- `StepConnector` 컴포넌트
