# Stepper

일정 개수의 steps의 progress를 나타냄

## Import

```js
import {
  Stepper,
  Step,
  StepDot,
  StepTitle,
  StepDescription
} from '@/components/navigation/Stepper';
```

- `Step` 컴포넌트
