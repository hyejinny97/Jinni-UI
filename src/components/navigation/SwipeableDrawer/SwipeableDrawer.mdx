# SwipeableDrawer

touch and drag(swipe)로 open/close가 가능한 Drawer

## Import

```js
import {
  SwipeableDrawer,
  SwipeableDrawerHeader,
  SwipeableDrawerBody,
  SwipeableDrawerFooter
} from '@/components/navigation/SwipeableDrawer';
```

## Usage

### Basic SwipeableDrawer

- `children`: swipeable drawer 콘텐츠 (`SwipeableDrawerHeader`, `SwipeableDrawerBody`, `SwipeableDrawerFooter` 등)
- `open`: true이면, swipeable drawer가 나타남
- `onOpen`: snap일 때 호출되는 함수
  - snap: 스와이프 동작을 멈췄을 때 drawer가 자동으로 특정 위치에 고정되거나 완전히 열림/닫힘 상태로 빠르게 이동하는 동작
- `onClose`: Escape 키/backdrop 클릭/snap일 때 호출되는 함수

### Snap points

- `snapPoints`: intersection ratio(0~1)에 따라 drawer가 어느 위치(0~1 비율)로 snap될지 정의
  - threshold: drawer의 현재 intersection ratio가 threshold 이하 가 되는 순간 snapTo가 적용됨
  - snapTo: drawer 길이 대비 비율 (0 = 닫힘, 1 = 열림)
  - snapPoints는 **위에서부터 순서대로 평가**되며, 첫 번째로 조건을 만족하는 snapPoint가 적용됨
  - snapTo가
    - 1 → onOpen 호출
    - 0 → onClose 호출
    - 0 < snapTo < 1 → 중간 위치 (onOpen/onClose 호출되지 않음)
