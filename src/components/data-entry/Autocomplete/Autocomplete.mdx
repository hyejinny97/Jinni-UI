# Autocomplete

입력창에 찾고자 하는 text 일부만 입력해도 여러 keyword options를 통해 빠르게 원하는 text 입력 가능하게 하는 컴포넌트

## Import

```js
import {
  Autocomplete,
  AutocompleteOption
} from '@/components/data-entry/Autocomplete';
```

## Usage

### Basic Autocomplete

- `Autocomplete` 컴포넌트
  - `children`: autocomplete의 options (AutocompleteOption 컴포넌트)
  - `defaultValue`: 초기 autocomplete value (uncontrolled 상태일 때만 적용 가능)
- `AutocompleteOption` 컴포넌트
  - `children`: option content
  - `value`: option의 value

<Canvas of={AutocompleteStories.BasicAutocomplete} />

### Controlled Autocomplete

#### 1) Control Autocomplete Value

- `value`와 `onChange` prop을 통해 Autocomplete Value를 control 가능
- `Autocomplete` 컴포넌트
  - `value`: autocomplete value
  - `onChange`: autocomplete value가 변경됐을 때 호출되는 함수

#### 2) Control Input Value

- `inputValue`와 `onInputChange` prop을 통해 Input Value를 control 가능
- `Autocomplete` 컴포넌트
  - `inputValue`: input value
  - `onInputChange`: input value가 변경됐을 때 호출되는 함수

<Canvas of={AutocompleteStories.ControlledAutocomplete} />

- `Autocomplete` 컴포넌트

- strict mode: autocomplete value는 options value만 가질 수 있음
