# Autocomplete

입력창에 찾고자 하는 text 일부만 입력해도 여러 keyword options를 통해 빠르게 원하는 text 입력 가능하게 하는 컴포넌트

## Import

```js
import {
  Autocomplete,
  AutocompleteOption
} from '@/components/data-entry/Autocomplete';
```

## Usage

### Basic Autocomplete

- `defaultValue`: 초기 autocomplete value (uncontrolled 상태일 때만 적용 가능)

<Canvas of={AutocompleteStories.BasicAutocomplete} />

### Controlled Autocomplete

#### 1. Control Autocomplete Value

- `value` prop을 통해 Autocomplete Value를 control 가능
- `value`: autocomplete value
- `onChange`: autocomplete value가 변경됐을 때 호출되는 함수

<Canvas of={AutocompleteStories.ControlledAutocompleteValue} />

#### 2. Control Input Value

- `inputValue` prop을 통해 Input Value를 control 가능
- `inputValue`: input value
- `onInputChange`: input value가 변경됐을 때 호출되는 함수

<Canvas of={AutocompleteStories.ControlledInputValue} />

#### 3. Control Menu

- `open` prop을 통해 `Menu`의 open state를 control 가능
- `open`: true이면, menu가 open 됨
- `onOpen`: open을 유발하는 이벤트가 발생 시 호출되는 함수
- `onClose`: close를 유발하는 이벤트가 발생 시 호출되는 함수

<Canvas of={AutocompleteStories.ControlledMenu} />

  - 'free' mode: autocomplete value는 options value 이외의 임의의 값도 가질 수 있음
