# TimePicker

시간 선택 가능

## Import

```js
import { TimePicker } from '@/components/data-entry/TimePicker';
```

## Usage

### Basic TimePicker

#### 1) mode

- `mode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식

<Canvas of={TimePickerStories.Mode} />

#### 2) defaultValue

- `defaultValue`: 초기 time (uncontrolled일 때만 적용 가능)

<Canvas of={TimePickerStories.DefaultValue} />

#### 3) form with timePicker

<Canvas of={TimePickerStories.FormWithTimePicker} />

### Controlled TimePicker

- `value`와 `onChange`를 통해 control 가능
- `value`: time
- `onChange`: value가 변경됐을 때 호출되는 함수

### Restrict Time

#### 1) minTime, maxTime

- `minTime`: 선택 가능한 최소 시간
- `maxTime`: 선택 가능한 최대 시간

<Canvas of={TimePickerStories.MinMaxTime} />

#### 2) disabledTimes

- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={TimePickerStories.DisabledTimes} />

### Time Steps

- `timeStep`: 두 time options 간 step
  - mode='preset'인 경우, `number` type (단위: s)
  - mode='manual'인 경우, `{hour: number, minute: number, second: number}` type
