# ColorBox

색상을 선택/변경할 수 있는 박스

## Import

```js
import { ColorBox } from '@/components/data-entry/ColorBox';
```

## Usage

### Basic ColorBox

- `defaultValue`: 초기 색상 (uncontrolled일 때만 적용 가능)

<Canvas of={ColorBoxStories.BasicColorBox} />

### Controlled ColorBox

- `value`와 `onChange` prop을 통해 control 가능
- `value`: 색상
- `onChange`: 색상(value)이 변경됐을 때 호출되는 함수

<Canvas of={ColorBoxStories.ControlledColorBox} />
