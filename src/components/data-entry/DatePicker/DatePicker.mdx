# DatePicker

날짜 선택 가능

## Import

```js
import { DatePicker } from '@/components/data-entry/DatePicker';
```

## Usage

### Basic DatePicker

- `defaultValue`: 초기 selected date (uncontrolled일 때만 적용 가능)

<Canvas of={DatePickerStories.BasicDatePicker} />

### Controlled DatePicker

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DatePickerStories.ControlledDatePicker} />
