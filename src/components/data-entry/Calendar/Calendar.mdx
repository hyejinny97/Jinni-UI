# Calendar

달력

## Import

```js
import { Calendar } from '@/components/data-entry/Calendar';
```

## Usage

### Basic Calendar

- `defaultValue`: 초기 selected date (uncontrolled일 때만 적용 가능)

<Canvas of={CalendarStories.BasicCalendar} />

### Controlled Calendar

- `value`와 `onChange`를 통해 control 가능
- `value`: date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={CalendarStories.ControlledCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={CalendarStories.ChooseTheInitialDate} />

### Locale and Options

- `locale`: BCP47 언어 태그를 포함하는 문자열
- `options`: date 속성

#### 1) Locale

<Canvas of={CalendarStories.Locale} />

#### 2) Options

<Canvas of={CalendarStories.Options} />

### Readonly

- `readOnly`: true이면, 수정 불가

<Canvas of={CalendarStories.Readonly} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={CalendarStories.Disabled} />

### Restrict Date

- `minDate`: 선택 가능한 최소 날짜
- `maxDate`: 선택 가능한 최대 날짜
- `disabledDates`: 비활성화하는 특정 날짜 모음

<Canvas of={CalendarStories.RestrictDate} />

### Order of years

- `yearsOrder`: years를 나타내는 순서

<Canvas of={CalendarStories.OrderOfYears} />

### Show additional days

- `showDaysOutsideCurrentMonth`: true이면, day calendar에서 해당 month의 days 이외의 앞뒤 일부 days를 나타냄
- `fixedWeekNumber`: day calendar에서 보여지는 week 개수

<Canvas of={CalendarStories.ShowAdditionalDays} />

### Display week number

- `displayWeekNumber`: true이면, week number가 나타남

<Canvas of={CalendarStories.DisplayWeekNumber} />

### Custom Day Calendar

- `renderDay`: dayProps를 인자로 받아 `Day` 컴포넌트를 반환하는 함수

#### 1) Week Picker

<Canvas of={CalendarStories.WeekPicker} />

#### 2) Calendar with notes

<Canvas of={CalendarStories.CalendarWithNotes} />

### Custom Month Calendar

- `renderMonth`: monthProps를 인자로 받아 `Month` 컴포넌트를 반환하는 함수

<Canvas of={CalendarStories.CustomMonthCalendar} />

### Custom Year Calendar

- `renderYear`: yearProps를 인자로 받아 `Year` 컴포넌트를 반환하는 함수

<Canvas of={CalendarStories.CustomYearCalendar} />
