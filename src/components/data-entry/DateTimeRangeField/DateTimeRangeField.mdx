# DateTimeRangeField

사용자가 선택한 날짜와 시간 범위가 나타나는 곳

## Import

```js
import { DateTimeRangeField } from '@/components/data-entry/DateTimeRangeField';
```

## Usage

### Basic DateTimeRangeField

- `placeholder`: placeholder
- `defaultValue`: 초기 datetime (uncontrolled일 때만 적용 가능)

<Canvas of={DateTimeRangeFieldStories.BasicDateTimeRangeField} />

### Controlled DateTimeRangeField

- `value`와 `onChange`를 통해 control 가능
- `value`: datetime
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateTimeRangeFieldStories.ControlledDateTimeRangeField} />

### Locale and Options

- `locale`: BCP47 언어 태그를 포함하는 문자열
- `options`: time 속성

#### 1) Locale

<Canvas of={DateTimeRangeFieldStories.Locale} />

#### 2) Options

<Canvas of={DateTimeRangeFieldStories.Options} />

### Customize DateTime Format

- `timeFormat`: time format

  - `locale`, `options` 대신 time tokens를 사용해 자유롭게 포맷 설정 가능

- `dateFormat`: date format

  - `locale`, `options` 대신 date tokens를 사용해 자유롭게 포맷 설정 가능

<Canvas of={DateTimeRangeFieldStories.CustomizeDateTimeFormat} />

### Restrict DateTime

#### 1) Restrict Date

- `minDate`: 선택 가능한 최소 날짜
- `maxDate`: 선택 가능한 최대 날짜
- `disabledDates`: 비활성화하는 특정 날짜 모음

<Canvas of={DateTimeRangeFieldStories.RestrictDate} />

#### 2) Restrict Time

- `minTime`: 선택 가능한 최소 시간
- `maxTime`: 선택 가능한 최대 시간
- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={DateTimeRangeFieldStories.RestrictTime} />

### Time Steps

- `timeMode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식
- `timeStep`: 두 time options 간 step
  - timeMode='preset'인 경우, `number` type (단위: s)
  - timeMode='manual'인 경우, `{hour: number, minute: number, second: number}` type
