# ManualDigitalClock

hour/minute/second를 각 선택 가능한 디지털 시계

## Import

```js
import { ManualDigitalClock } from '@/components/data-entry/ManualDigitalClock';
```

## Usage

### Basic ManualDigitalClock

- `defaultValue`: 초기 selected time (uncontrolled일 때만 적용 가능)

<Canvas of={ManualDigitalClockStories.BasicManualDigitalClock} />

### Controlled ManualDigitalClock

- `value`를 통해 control 가능
- `value`: selected time
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={ManualDigitalClockStories.ControlledManualDigitalClock} />

### Locale and Options

#### 1. Locale

- `locale`: BCP47 언어 태그를 포함하는 문자열

<Canvas of={ManualDigitalClockStories.Locale} />

#### 2. Options

- `options`: time 속성

<Canvas of={ManualDigitalClockStories.Options} />

### Restrict Time

#### 1. Min time

- `minTime`: 선택 가능한 최소 시간

<Canvas of={ManualDigitalClockStories.MinTime} />

#### 2. Max time

- `maxTime`: 선택 가능한 최대 시간

<Canvas of={ManualDigitalClockStories.MaxTime} />

#### 3. Disabled times

- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={ManualDigitalClockStories.DisabledTimes} />

#### 4. Skip disabled time

- `skipDisabledTime`: true이면, 비활성화된 times는 나타나지 않음

<Canvas of={ManualDigitalClockStories.SkipDisabledTime} />

### Time Steps

- `timeStep`: 두 time options 간 step

<Canvas of={ManualDigitalClockStories.TimeStep} />

### Readonly

- `readOnly`: true이면, 수정 불가

<Canvas of={ManualDigitalClockStories.Readonly} />

### Disabled

- `disabled`: true이면, 비활성화됨
