# FileInput

files를 선택할 수 있음

## Import

```js
import { FileInput } from '@/components/data-entry/FileInput';
```

## Usage

### Basic FileInput

- `children`: file input을 나타내는 요소
- `value`: file input의 value
- `onChange`: file input의 value가 변경됐을 때 호출되는 함수
  - `event.target.files`를 통해 새로 선택된 value 값을 얻을 수 있지만, 두 번째 인자인 value 값을 사용하는 것을 추천함

### Multiple

- `multiple`: true이면, 2개 이상의 파일을 받을 수 있음
- 기본적으로, `value` prop에는 중복되는 파일이 존재할 수 있음
