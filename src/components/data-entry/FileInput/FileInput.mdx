# FileInput

files를 선택할 수 있음

## Import

```js
import { FileInput } from '@/components/data-entry/FileInput';
```

## Usage

### Basic FileInput

#### 1) Various FileInput Appearances

- `children`: file input을 나타내는 요소
- `value`: file input의 value
  - type: multiple === true ? `Array<File>` : `File` | `null`
- `onChange`: file input의 value가 변경됐을 때 호출되는 함수
  - event: `event.target.files`를 통해 새로 선택된 value 값을 얻을 수 있음
    - type: `FileList` | `null`
  - value: file input의 new value
    - type: multiple === true ? `Array<File>` : `File` | `null`
    - value = 기존 value 값 + 새로 선택된 value 값
