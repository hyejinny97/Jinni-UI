# TextArea

여러 줄의 text를 입력하거나 수정할 수 있는 필드

## Import

```js
import { TextArea } from '@/components/data-entry/TextArea';
```

## Usage

### Basic TextArea

- `defaultValue`: 초기 textarea value (uncontrolled 상태일 때만 적용 가능)

<Canvas of={TextAreaStories.BasicTextArea} />

### Controlled TextArea

- `value`와 `onChange` prop을 통해 control 가능
- `value`: textarea value
- `onChange`: value가 변경됐을 때 호출되는 함수
  - `event.target.value`를 통해 new value 접근 가능

<Canvas of={TextAreaStories.ControlledTextArea} />

### Rows

- 기본적으로, textarea는 한 줄이고 라인 수가 증가할 때마다 높이가 자동으로 일정하게 증가함
- `rows`: rows 개수
  - 이 속성을 설정하면, `maxRows`나 `minRows` 값을 지정해도 반영이 안되고 줄 수(높이)가 고정됨
- `maxRows`: 최대 rows 개수
  - `rows` 개수가 `maxRows` 이하라면, textarea는 라인 수가 증가할 때마다 높이가 자동으로 일정하게 증가함
- `minRows`: 최소 rows 개수
  - `rows` 개수가 `minRows` 이상라면, textarea는 라인 수가 증가할 때마다 높이가 자동으로 일정하게 증가함
