# DateTimePicker

날짜와 시간 선택 가능

## Import

```js
import { DateTimePicker } from '@/components/data-entry/DateTimePicker';
```

## Usage

### Basic DateTimePicker

- `defaultValue`: 초기 selected date time (uncontrolled일 때만 적용 가능)

<Canvas of={DateTimePickerStories.BasicDateTimePicker} />

### Controlled DateTimePicker

- `value`를 통해 control 가능
- `value`: selected date time
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateTimePickerStories.ControlledDateTimePicker} />

### Time Mode

- `timeMode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식

<Canvas of={DateTimePickerStories.TimeMode} />

### Time Steps

- `timeStep`: 두 time options 간 step
  - timeMode='preset'인 경우, `number` type (단위: s)
  - timeMode='manual'인 경우, `{hour: number, minute: number, second: number}` type
