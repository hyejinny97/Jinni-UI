# Radio

여러 개의 옵션 중에 하나만 선택 가능하게 함

## Import

```js
import { Radio } from '@/components/data-entry/Radio';
```

## Usage

### Basic Radio

- type='radio'인 input 요소의 props 전부 가능 (ex) name, value, required 등)
- `disableRipple`: true이면, ripple effect가 사라짐

<Canvas of={RadioStories.BasicRadio} />

### Checked by default

- `defaultChecked`: true이면, 기본적으로 check 됨 (uncontrolled 상태일 때 적용 가능)

<Canvas of={RadioStories.CheckedByDefault} />

### Disabled

- `disabled`: true이면, 비활성화 됨

<Canvas of={RadioStories.Disabled} />

### Size

- `size`: radio 크기

<Canvas of={RadioStories.Size} />

### Color

- `color`: radio 색상

<Canvas of={RadioStories.Color} />

### Icon

- `icon`: unchecked 됐을 때 나타나는 icon
- `checkedIcon`: checked 됐을 때 나타나는 icon

<Canvas of={RadioStories.Icon} />

### Controlled Radio

- `checked` 와 `onChange` prop을 통해 radio를 control 가능
- `checked`: true이면, check 됨
- `onChange`: checked/unchecked state가 변경됐을 때 호출되는 함수

<Canvas of={RadioStories.ControlledRadio} />
