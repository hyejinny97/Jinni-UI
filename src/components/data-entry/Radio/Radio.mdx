# Radio

여러 개의 옵션 중에 하나만 선택 가능하게 함

## Import

```js
import { Radio } from '@/components/data-entry/Radio';
```

## Usage

### Controlled Radio

- `checked` prop을 통해 radio를 control 가능
- `checked`: true이면, check 됨
- `onChange`: checked state가 변경됐을 때 호출되는 함수
  - `event.target.checked` (boolean)을 통해 new checked state에 접근 가능
  - `event.target.value` (string)을 통해 new value에 접근 가능

### Customize Icon

- `icon`: unchecked 됐을 때 나타나는 icon
- `checkedIcon`: checked 됐을 때 나타나는 icon

- type='radio'인 input 요소의 props 전부 가능
