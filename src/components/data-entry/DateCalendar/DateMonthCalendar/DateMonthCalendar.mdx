# DateMonthCalendar

월별 달력

## Import

```js
import { DateMonthCalendar } from '@/components/data-entry/DateCalendar';
```

## Usage

### Basic DateMonthCalendar

- `defaultValue`: 초기 selected date

<Canvas of={DateMonthCalendarStories.BasicDateMonthCalendar} />

### Controlled DateMonthCalendar

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateMonthCalendarStories.ControlledDateMonthCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={DateMonthCalendarStories.ChooseTheInitialDate} />

<Canvas of={DateMonthCalendarStories.Options} />

#### 1. Custom Month
