## Import

```js
import { DateDayCalendar } from '@/components/data-entry/DateCalendar';
```

## Usage

### Basic DateDayCalendar

- `defaultValue`: 초기 selected date

<Canvas of={DateDayCalendarStories.BasicDateDayCalendar} />

### Controlled DateDayCalendar

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateDayCalendarStories.ControlledDateDayCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={DateDayCalendarStories.ChooseTheInitialDate} />
