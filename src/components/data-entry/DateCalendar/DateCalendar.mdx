```js
import { DateCalendar } from '@/components/data-entry/DateCalendar';
```

## Usage

### Basic DateCalendar

- `defaultValue`: 초기 selected date

<Canvas of={DateCalendarStories.BasicDateCalendar} />

### Controlled DateCalendar

- `value`를 통해 control 가능

- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateCalendarStories.ControlledDateCalendar} />
