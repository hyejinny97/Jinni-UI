# Month

month 버튼

## Import

```js
import { Month } from '@/components/data-entry/MonthCalendar';
```

## Usage

### Basic Month

- `value`: 다른 Month 컴포넌트와 구별되는 식별자
- `children`: button 콘텐츠

<Canvas of={MonthStories.BasicMonth} />

### Selected Month

- `selected`: true이면, selected style이 나타남

<Canvas of={MonthStories.SelectedMonth} />

### Marked Month

- `marked`: true이면, marked style이 나타남

<Canvas of={MonthStories.MarkedMonth} />

### Color

- `color`: selected 됐을 때 나타나는 배경색

<Canvas of={MonthStories.Color} />

### ReadOnly

- `readOnly` : true이면, click handler가 호출되지 않음

<Canvas of={MonthStories.ReadOnly} />

### ButtonBase props

- `Month` 컴포넌트는 내부적으로 `ButtonBase` 컴포넌트를 반환하므로, `ButtonBase` 의 모든 props 사용 가능
