# MonthCalendar

month 달력

## Import

```js
import { MonthCalendar } from '@/components/data-entry/MonthCalendar';
```

## Usage

### Basic MonthCalendar

- `displayedDate`: 화면에 보여지는 날짜

<Canvas of={MonthCalendarStories.BasicMonthCalendar} />

### Selected date

- `selectedDate`: 선택된 날짜
- `onMonthChange`: month button을 클릭했을 때 호출되는 함수

<Canvas of={MonthCalendarStories.SelectedDate} />

### Locale

- `locale`: BCP47 언어 태그를 포함하는 문자열

<Canvas of={MonthCalendarStories.Locale} />

### Restrict Date

#### 1. Min date

- `minDate`: 선택 가능한 최소 날짜

<Canvas of={MonthCalendarStories.MinDate} />

#### 2. Max date

- `maxDate`: 선택 가능한 최대 날짜

<Canvas of={MonthCalendarStories.MaxDate} />

### ReadOnly

- `readOnly`: true이면, 수정 불가

<Canvas of={MonthCalendarStories.ReadOnly} />

- `disabled`: true이면, 비활성화됨

<Canvas of={MonthCalendarStories.Disabled} />
