# DateRangePicker

날짜 범위 선택 가능

## Import

```js
import { DateRangePicker } from '@/components/data-entry/DateRangePicker';
```

## Usage

### Basic DateRangePicker

- `defaultValue`: 초기 selected date (uncontrolled일 때만 적용 가능)

<Canvas of={DateRangePickerStories.BasicDateRangePicker} />

### Controlled DateRangePicker

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateRangePickerStories.ControlledDateRangePicker} />
