# CheckboxLabel

checkbox의 label

## Import

```js
import { CheckboxLabel } from '@/components/data-entry/CheckboxLabel';
```

## Usage

### Basic CheckboxLabel

- label 요소의 props 전부 가능 (ex) form, htmlFor 등)
- `children`: Checkbox 컴포넌트
- `label`: label text or element

<Canvas of={CheckboxLabelStories.BasicCheckboxLabel} />

### Required

- `required`: true이면, input을 required 처리하고 label 마지막에 ‘\*’ 표시 추가

<Canvas of={CheckboxLabelStories.Required} />

### Disabled

- `disabled`: true이면 input을 disabled 처리하고 label 색상을 dim 처리함

<Canvas of={CheckboxLabelStories.Disabled} />

### Label Placement

- `labelPlacement`: label 위치

<Canvas of={CheckboxLabelStories.LabelPlacement} />

### Customization

- `style` prop을 통해 gap, font-size, font-color 등 커스텀 가능
