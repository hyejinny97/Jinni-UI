# Year

year 버튼

## Import

```js
import { Year } from '@/components/data-entry/YearCalendar';
```

## Usage

### Basic Year

- `value`: 다른 Year 컴포넌트와 구별되는 식별자
- `children`: button 콘텐츠

<Canvas of={YearStories.BasicYear} />

### Selected Year

- `selected`: true이면, selected style이 나타남

<Canvas of={YearStories.SelectedYear} />

### Marked Year

- `marked`: true이면, marked style이 나타남

<Canvas of={YearStories.MarkedYear} />

### Color

- `color`: selected 됐을 때 나타나는 배경색

<Canvas of={YearStories.Color} />

### ReadOnly

- `readOnly` : true이면, click handler가 호출되지 않음

<Canvas of={YearStories.ReadOnly} />

### ButtonBase props

- `Year` 컴포넌트는 내부적으로 `ButtonBase` 컴포넌트를 반환하므로, `ButtonBase` 의 모든 props 사용 가능

#### 1. Overlay Effect

#### 2. Ripple Effect

<Canvas of={YearStories.RippleEffect} />

#### 3. Elevation Effect

<Canvas of={YearStories.ElevationEffect} />

<Canvas of={YearStories.Disabled} />
