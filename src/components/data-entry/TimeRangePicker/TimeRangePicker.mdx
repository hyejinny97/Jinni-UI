# TimeRangePicker

시간 범위 선택 가능

## Import

```js
import { TimeRangePicker } from '@/components/data-entry/TimeRangePicker';
```

## Usage

### Basic TimeRangePicker

#### 1) mode

- `mode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식

<Canvas of={TimeRangePickerStories.Mode} />

#### 2) defaultValue

- `defaultValue`: 초기 time (uncontrolled일 때만 적용 가능)

<Canvas of={TimeRangePickerStories.DefaultValue} />

#### 3) form with timeRangePicker

<Canvas of={TimeRangePickerStories.FormWithTimeRangePicker} />

### Controlled TimeRangePicker

- `value`와 `onChange`를 통해 control 가능
- `value`: time
- `onChange`: value가 변경됐을 때 호출되는 함수

### Restrict Time

#### 1) minTime, maxTime

- `minTime`: 선택 가능한 최소 시간
- `maxTime`: 선택 가능한 최대 시간

<Canvas of={TimeRangePickerStories.MinMaxTime} />

#### 2) disabledTimes

- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={TimeRangePickerStories.DisabledTimes} />

### Time Steps

- `timeStep`: 두 time options 간 step
  - mode='preset'인 경우, `number` type (단위: s)
  - mode='manual'인 경우, `{hour: number, minute: number, second: number}` type
