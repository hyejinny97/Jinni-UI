# DateTimeRangePicker

날짜와 시간 범위 선택 가능

## Import

```js
import { DateTimeRangePicker } from '@/components/data-entry/DateTimeRangePicker';
```

## Usage

### Basic DateTimeRangePicker

#### 1) Basic

- `defaultValue`: 초기 datetime (uncontrolled일 때만 적용 가능)

<Canvas of={DateTimeRangePickerStories.Basic} />

#### 2) Time Mode

- `timeMode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식

<Canvas of={DateTimeRangePickerStories.TimeMode} />

#### 3) Form with DateTimeRangePicker

<Canvas of={DateTimeRangePickerStories.FormWithDateTimeRangePicker} />

### Controlled DateTimeRangePicker

- `value`와 `onChange`를 통해 control 가능
- `value`: datetime
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateTimeRangePickerStories.ControlledDateTimeRangePicker} />

- `options`: date 속성

- `readOnly`: true이면, 수정 불가

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={DateTimeRangePickerStories.Disabled} />

### Restrict DateTime

#### 1) Restrict Date

- `minDate`: 선택 가능한 최소 날짜
- `maxDate`: 선택 가능한 최대 날짜
- `disabledDates`: 비활성화하는 특정 날짜 모음

<Canvas of={DateTimeRangePickerStories.RestrictDate} />

#### 2) Restrict Time

- `minTime`: 선택 가능한 최소 시간
- `maxTime`: 선택 가능한 최대 시간
- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={DateTimeRangePickerStories.RestrictTime} />

### Time Steps

- `timeStep`: 두 time options 간 step
  - timeMode='preset'인 경우, `number` type (단위: s)
  - timeMode='manual'인 경우, `{hour: number, minute: number, second: number}` type
