# DateTimeField

사용자가 선택한 시간과 날짜가 나타나는 곳

## Import

```js
import { DateTimeField } from '@/components/data-entry/DateTimeField';
```

## Usage

- `placeholder`: placeholder
- `defaultValue`: 초기 date time (uncontrolled일 때만 적용 가능)

### DateTime Format

#### 1. Date format

- `dateFormat`: date format

  - `options` 대신 date tokens를 사용해 자유롭게 포맷 설정 가능
  - `format`과 `option`(DateOptions) prop 동시 사용 시, `option` prop의 값은 무시되고 `format` prop 값이 적용됨

### Time Steps

- `timeMode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식
- `timeStep`: 두 time options 간 step
  - timeMode='preset'인 경우, `number` type (단위: s)
  - timeMode='manual'인 경우, `{hour: number, minute: number, second: number}` type
