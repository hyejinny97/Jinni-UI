# DateRangeField

사용자가 선택한 날짜 범위가 나타나는 곳

## Import

```js
import { DateRangeField } from '@/components/data-entry/DateRangeField';
```

## Usage

### Basic DateRangeField

- `placeholder`: placeholder
- `defaultValue`: 초기 date (uncontrolled일 때만 적용 가능)

<Canvas of={DateRangeFieldStories.BasicDateRangeField} />

### Controlled DateRangeField

- `value`를 통해 control 가능
- `value`: date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateRangeFieldStories.ControlledDateRangeField} />

### Locale and Options

#### 1. Locale

- `locale`: BCP47 언어 태그를 포함하는 문자열

<Canvas of={DateRangeFieldStories.Locale} />

#### 2. Options

- `options`: date 속성

<Canvas of={DateRangeFieldStories.Options} />

### Date Format

- `format`: date format

  - `options` 대신 date tokens를 사용해 자유롭게 포맷 설정 가능
  - `format`과 `option` prop 동시 사용 시, `option` prop의 값은 무시되고 `format` prop 값이 적용됨

<details>
  <summary>Date Tokens</summary>
  <Markdown>
    {`
| Token | Type | 특징 | 
| ----- | --------- | ---------------- | 
| YYYY | year | 'numeric' | 
| YY | year | '2-digit' | 
| MMMM | month | 'long' | 
| MMM | month | 'short' | 
| MM | month | '2-digit' | 
| M | month | 'numeric' | 
| DD | day | '2-digit' | 
| D | day | 'numeric' | 
`}
  </Markdown>
</details>
