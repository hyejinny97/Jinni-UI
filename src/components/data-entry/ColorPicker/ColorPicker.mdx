# ColorPicker

색상 선택이 가능한 컴포넌트

## Import

```js
import {
  ColorPicker,
  ColorBox,
  ColorField
} from '@/components/data-entry/ColorPicker';
```

## Usage

### Basic ColorPicker

- `defaultValue`: 초기 색상 (uncontrolled일 때만 적용 가능)

<Canvas of={ColorPickerStories.BasicColorPicker} />

### Controlled ColorPicker

- `value` prop을 통해 control 가능
- `value`: 색상
- `onChange`: 색상이 변경됐을 때 호출되는 함수

<Canvas of={ColorPickerStories.ControlledColorPicker} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={ColorPickerStories.Disabled} />

### ColorPicker with form

<Canvas of={ColorPickerStories.ColorPickerWithForm} />

### Custom ColorField

- `renderColorField`: colorFieldProps를 입력값으로 받아 `ColorField` 컴포넌트를 렌더하는 함수

<Canvas of={ColorPickerStories.CustomColorField} />

### Custom ColorBox

- `renderColorBox`: colorBoxProps를 입력값으로 받아 `ColorBox` 컴포넌트를 렌더하는 함수

<Canvas of={ColorPickerStories.CustomColorBox} />

### Custom Popover
