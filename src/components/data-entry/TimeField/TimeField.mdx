# TimeField

사용자가 선택한 시간이 나타나는 곳

## Import

```js
import { TimeField } from '@/components/data-entry/TimeField';
```

## Usage

### Basic TimeField

- `placeholder`: placeholder
- `defaultValue`: 초기 time (uncontrolled일 때만 적용 가능)

<Canvas of={TimeFieldStories.BasicTimeField} />

### Controlled TimeField

- `value`와 `onChange`를 통해 control 가능
- `value`: time
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={TimeFieldStories.ControlledTimeField} />

- `locale`: BCP47 언어 태그를 포함하는 문자열
- `options`: time 속성

### Customize Time Format

- `format`: time format

  - `locale`, `options` 대신 time tokens를 사용해 자유롭게 포맷 설정 가능

- Time Tokens

  <Markdown>
    {`

  | ----- | --------- | ---------------- | 
  | HH | hour | 'h23', '2-digit' | 
  | H | hour | 'h23', 'numeric' | 
  | hh | hour | 'h11', '2-digit' | 
  | h | hour | 'h11', 'numeric' | 
  | TT | hour | 'h24', '2-digit' | 
  | T | hour | 'h24', 'numeric' | 
  | tt | hour | 'h12', '2-digit' |

  | mm | minute | '2-digit' | 
  | m | minute | 'numeric' | 
  | ss | second | '2-digit' |

  | a | dayPeriod | 'lowerCase' | 
  | A | dayPeriod | 'upperCase' |
  `}

### Restrict Time

- `minTime`: 선택 가능한 최소 시간
- `maxTime`: 선택 가능한 최대 시간
- `disabledTimes`: 비활성화하는 특정 시간 모음

<Canvas of={TimeFieldStories.RestrictTime} />

### Time Steps

- `mode`: time 선택 방법
  - `'preset'`: '12:00', '12:30', '01:00' 처럼 미리 정의된 시간 목록에서 하나를 선택하는 방식
  - `'manual'`: 시(hour), 분(minute), 초(second)를 각각 선택하는 방식
- `timeStep`: 두 time options 간 step
  - mode='preset'인 경우, `number` type (단위: s)
  - mode='manual'인 경우, `{hour: number, minute: number, second: number}` type
