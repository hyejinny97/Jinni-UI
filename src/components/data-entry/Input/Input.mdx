# Input

string이나 number를 입력하거나 수정할 수 있는 input

## Import

```js
import { Input } from '@/components/data-entry/Input';
```

## Usage

### Basic Input

- `type`: input 타입
- `defaultValue`: 초기 input value (uncontrolled 상태일 때만 적용 가능)

<Canvas of={InputStories.BasicInput} />

### Controlled Input

- `value` prop을 통해 control 가능
- `value`: input value
- `onChange`: value가 변경됐을 때 호출되는 함수
  - `event.target.value`를 통해 new value 접근 가능

<Canvas of={InputStories.ControlledInput} />
