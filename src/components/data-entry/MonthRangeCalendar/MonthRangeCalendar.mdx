# MonthRangeCalendar

month range 달력

## Import

```js
import { MonthRangeCalendar } from '@/components/data-entry/MonthRangeCalendar';
```

## Usage

### Basic MonthRangeCalendar

- `displayedDate`: 화면에 보여지는 날짜

<Canvas of={MonthRangeCalendarStories.BasicMonthRangeCalendar} />

### Controlled select

- `selectedDate`를 통해 select range를 control 가능
- `selectedDate`: 선택된 날짜 범위
- `onSelectDate`: selected date가 변경되었을 때 호출되는 함수

<Canvas of={MonthRangeCalendarStories.ControlledSelect} />

### Controlled hover

- `hoveredDate`를 통해 hover range를 control 가능
- `hoveredDate`: hover한 날짜
- `onHoverDate`: 날짜에 hover 했을 때 호출되는 함수

<Canvas of={MonthRangeCalendarStories.ControlledHover} />

### Locale

- `locale`: BCP47 언어 태그를 포함하는 문자열

<Canvas of={MonthRangeCalendarStories.Locale} />

### Restrict Date

#### 1. Min date

- `minDate`: 선택 가능한 최소 날짜

<Canvas of={MonthRangeCalendarStories.MinDate} />

#### 2. Max date

- `maxDate`: 선택 가능한 최대 날짜

<Canvas of={MonthRangeCalendarStories.MaxDate} />

### ReadOnly

- `readOnly`: true이면, 수정 불가

<Canvas of={MonthRangeCalendarStories.ReadOnly} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={MonthRangeCalendarStories.Disabled} />
