# ColorField

사용자가 선택한 색상을 보여주는 영역

## Import

```js
import { ColorField } from '@/components/data-entry/ColorField';
```

## Usage

### Basic ColorField

<Canvas of={ColorFieldStories.BasicColorField} />

### Color Value

- `value`: 색상
  - type: `ColorType`
    - CSSColorKeywords
    - HEX (#RRGGBB[AA], #RGB[A])
    - RGB, RGBA
    - HSL, HSLA
    - JinniColorTheme, JinniColorPalette
    - 'transparent'

<Canvas of={ColorFieldStories.ColorValue} />

### Variants

- `variant`: input base 형태

<Canvas of={ColorFieldStories.Variants} />

### Sizes

- `size`: input base 크기

<Canvas of={ColorFieldStories.Sizes} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={ColorFieldStories.Disabled} />

### Color

- `color`: base color
- `focusedColor`: focused color

<Canvas of={ColorFieldStories.Color} />

### Full Width

- `fullWidth`: true이면, input이 container 안에 꽉 채워지게 됨

<Canvas of={ColorFieldStories.FullWidth} />

### Adornment

- `startAdornment`: input base 앞에 위치하는 부가 요소 (icon ,text 등)
- `endAdornment`: input base 뒤에 위치하는 부가 요소 (icon ,text 등)

<Canvas of={ColorFieldStories.Adornment} />

### Disable Effect

- `disableHoverEffect`: true이면, hover effect가 비활성화됨
- `disableFocusEffect`: true이면, focus effect가 비활성화됨

<Canvas of={ColorFieldStories.DisableEffect} />

### Custom Color Field

- `children`: color field content

<Canvas of={ColorFieldStories.CustomColorField} />
