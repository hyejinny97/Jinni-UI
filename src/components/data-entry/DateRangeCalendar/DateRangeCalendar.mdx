# DateRangeCalendar

날짜 범위 선택 가능한 달력

## Import

```js
import { DateRangeCalendar } from '@/components/data-entry/DateRangeCalendar';
```

## Usage

### Basic DateRangeCalendar

- `defaultValue`: 초기 selected date

<Canvas of={DateRangeCalendarStories.BasicDateRangeCalendar} />

### Controlled DateRangeCalendar

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={DateRangeCalendarStories.ControlledDateRangeCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={DateRangeCalendarStories.ChooseTheInitialDate} />
