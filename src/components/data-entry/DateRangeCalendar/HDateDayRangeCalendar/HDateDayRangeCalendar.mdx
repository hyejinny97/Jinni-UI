# HDateDayRangeCalendar

day range 달력

## Import

```js
import { HDateDayRangeCalendar } from '@/components/data-entry/DateRangeCalendar';
```

## Usage

### Basic HDateDayRangeCalendar

- `defaultValue`: 초기 selected date (uncontrolled일 때만 적용 가능)

<Canvas of={HDateDayRangeCalendarStories.BasicHDateDayRangeCalendar} />

### Controlled HDateDayRangeCalendar

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={HDateDayRangeCalendarStories.ControlledHDateDayRangeCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={HDateDayRangeCalendarStories.ChooseInitialDate} />

#### 1. Locale

<Canvas of={HDateDayRangeCalendarStories.Locale} />

#### 2. Options

<Canvas of={HDateDayRangeCalendarStories.Options} />
