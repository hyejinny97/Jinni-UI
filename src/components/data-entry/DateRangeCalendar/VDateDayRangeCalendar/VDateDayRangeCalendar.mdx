# VDateDayRangeCalendar

day range 달력

## Import

```js
import { VDateDayRangeCalendar } from '@/components/data-entry/DateRangeCalendar';
```

## Usage

### Basic VDateDayRangeCalendar

- `defaultValue`: 초기 selected date (uncontrolled일 때만 적용 가능)

<Canvas of={VDateDayRangeCalendarStories.BasicVDateDayRangeCalendar} />

### Controlled VDateDayRangeCalendar

- `value`를 통해 control 가능
- `value`: selected date
- `onChange`: value가 변경됐을 때 호출되는 함수

<Canvas of={VDateDayRangeCalendarStories.ControlledVDateDayRangeCalendar} />

### Choose the initial date

- `referenceDate`: `value`, `defaultValue` prop이 없을 때 참조하는 date

<Canvas of={VDateDayRangeCalendarStories.ChooseInitialDate} />

#### 1. Locale

- `locale`: BCP47 언어 태그를 포함하는 문자열

<Canvas of={VDateDayRangeCalendarStories.Locale} />

#### 2. Options

- `options`: date 속성

<Canvas of={VDateDayRangeCalendarStories.Options} />
