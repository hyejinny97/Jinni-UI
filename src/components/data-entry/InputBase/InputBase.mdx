# InputBase

input UI를 가지는 컴포넌트
(내부에 HTML input element는 없고 오로지 input UI만 가짐)

## Import

```js
import { InputBase } from '@/components/data-entry/InputBase';
```

## Usage

### Basic InputBase

- `children`: input content

<Canvas of={InputBaseStories.BasicInputBase} />

### Variants

- `variant`: input 형태

<Canvas of={InputBaseStories.Variants} />

### Sizes

- `size`: input 크기

<Canvas of={InputBaseStories.Sizes} />

### Disabled

- `disabled`: true이면, 비활성화됨

<Canvas of={InputBaseStories.Disabled} />

### Color

- `color`: base color
- `focusedColor`: focused color

<Canvas of={InputBaseStories.Color} />

### Full Width

- `fullWidth`: true이면, input이 container 안에 꽉 채워지게 됨

<Canvas of={InputBaseStories.FullWidth} />

### Adornments

- `startAdornment`: input 앞에 위치하는 부가 요소 (icon ,text 등)
- `endAdornment`: input 뒤에 위치하는 부가 요소 (icon ,text 등)

<Canvas of={InputBaseStories.Adornments} />

### Disable Effect

- `disableHoverEffect`: true이면, hover effect가 비활성화됨
- `disableFocusEffect`: true이면, focus effect가 비활성화됨

<Canvas of={InputBaseStories.DisableEffect} />
