# Modal

중요한 정보를 포함하거나, 결정을 요청하거나, 복잡한 일 처리를 포함함

## Import

```js
import {
  Modal,
  ModalHeader,
  ModalBody,
  ModalFooter
} from '@/components/feedback/Modal';
```

## Usage

### Basic Modal

- `children`: modal 콘텐츠 (ModalHeader, ModalBody, ModalFooter 등)
- `open`: true이면, modal이 나타남
- `onClose`: Escape 키/backdrop 클릭 이벤트가 일어날 때 호출되는 함수

<Canvas of={ModalStories.BasicModal} />

### Sizes

- `size`: modal의 너비

<Canvas of={ModalStories.Sizes} />

### Scrolling Long Content

- `scrollBehavior`: modal 내 content가 뷰포트를 넘길 때 scroll 하는 방법

  - `inside`: `ModalBody`가 scroll 됨
  - `outside`: `ModalContent`(=전체)가 scroll 됨
