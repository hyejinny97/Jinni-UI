# Modal

중요한 정보를 포함하거나, 결정을 요청하거나, 복잡한 일 처리를 포함함

## Import

```js
import {
  Modal,
  ModalHeader,
  ModalBody,
  ModalFooter
} from '@/components/feedback/Modal';
```

## Usage

### Basic Modal

- `Modal` 컴포넌트

  - `children`: Modal 내 ModalContent(=Box) 컴포넌트에 담겨질 요소
    - `ModalHeader`, `ModalBody`, `ModalFooter` 등
  - `open`: true이면, modal이 나타남
  - `onClose`: "escapeKeyDown", "backdropClick" 이벤트가 일어날 때 호출되는 함수

- `ModalHeader` 컴포넌트

- `ModalBody` 컴포넌트

- `ModalFooter` 컴포넌트

### Sizes

- `size`: modal의 너비
  - `xs` | `sm` | `md`(default) | `lg` | `xl` | `full` | `Responsive<xs | sm | md | lg | xl | full>`

### Scrolling Long Content

- `scrollBehavior`: modal 내 content가 뷰포트를 넘길 때 scroll 하는 방법

  - `inside`(default): `ModalBody`가 scroll 됨
  - `outside`: `ModalContent`(=전체)가 scroll 됨
