# CircularProgress

명확하지 않은 대기 시간을 알려주거나 처리 정도를 나타내줌

## Import

```js
import { CircularProgress } from '@/components/feedback/CircularProgress';
```

## Usage

### Basic CircularProgress

- 종류

  - indeterminate: 명확하지 않은 대기 상태 (shrink animation이 있음)
  - determinate: 명확한 대기 상태 (처리 정도를 나타내줌)

- `percent`: 처리 정도
  - `percent`가 있으면 determinate, 없으면 indeterminate
