# CircularProgress

명확하지 않은 대기 시간을 알려주거나 처리 정도를 나타내줌

- CircularProgress 종류
  - Indeterminate CircularProgress: 명확하지 않은 대기 상태를 나타냄
  - Determinate CircularProgress: 명확한 진행 정도를 나타냄

## Import

```js
import { CircularProgress } from '@/components/feedback/CircularProgress';
```
