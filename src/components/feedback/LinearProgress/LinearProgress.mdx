# LinearProgress

명확하지 않은 대기 시간을 알려주거나 처리 정도를 나타내줌

- LinearProgress 종류
  - Indeterminate LinearProgress: 명확하지 않은 대기 상태를 나타냄
  - Determinate LinearProgress: 명확한 진행 정도를 나타냄

## Import

```js
import { LinearProgress } from '@/components/feedback/LinearProgress';
```
