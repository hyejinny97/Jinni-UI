# LinearProgress

명확하지 않은 대기 시간을 알려주거나 처리 정도를 나타내줌

## Import

```js
import { LinearProgress } from '@/components/feedback/LinearProgress';
```

## Usage

### Basic LinearProgress

- 종류

  - indeterminate: 명확하지 않은 대기 상태
  - determinate: 명확한 대기 상태 (처리 정도를 나타내줌)

- `percent`: 처리 정도
  - `percent`가 있으면 determinate, 없으면 indeterminate
