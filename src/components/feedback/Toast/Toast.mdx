# Toast

간단한 처리 정도를 알려줌

## Import

```js
import { Toast } from '@/components/feedback/Toast';
```

## Usage

### Basic Toast

- `message`: toast 콘텐츠
- `open`: true이면, toast가 나타남
- `onClose`: 타임아웃/Escape 키/background 클릭 이벤트 발생 시 호출되는 함수

<Canvas of={ToastStories.BasicToast} />

### Toast Action

- `action`: toast 끝에 나타나는 action

<Canvas of={ToastStories.ToastAction} />

### Toast Position

- `anchorOrigin`: toast가 위치할 anchor의 origin

<Canvas of={ToastStories.ToastPosition} />

### Automatic Dismiss

- `autoHideDuration`: `onClose` 함수를 자동으로 호출하기 전까지 기다리는 시간 (단위: s)
  - null이면, 자동으로 `onClose` 함수를 호출하지 않음
  - toast에 hover하고 있는 동안의 시간은 카운트 되지 않음

<Canvas of={ToastStories.AutomaticDismiss} />

### Use with Alerts

- `children`: 기존 `Box`를 대체할 요소

<Canvas of={ToastStories.UseWithAlerts} />

### Consecutive Toast

- `key`를 통해 하나의 Toast 컴포넌트로 여러 개의 toast를 나타낼 수 있음

<Canvas of={ToastStories.ConsecutiveToastWithoutStacking} />

### Customize Toast

- `BoxProps`: `Box` 컴포넌트에 적용되는 props
- `style`을 통해 padding 등 커스텀 가능

<Canvas of={ToastStories.CustomizeToast} />

### Customize Transition

- `TransitionComponent`: transition 컴포넌트
