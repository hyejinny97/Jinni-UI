# Tree

데이터를 계층적으로 나타냄

## Import

```js
import { Tree, TreeItem } from '@/components/data-display/Tree';
```

## Usage - Basic

### Basic Tree

- `data`: tree item에 관한 정보
- `treeNodes`: tree 구조
- 주의할 점) data와 treeNodes 값이 변경되면 전체 구조를 재생성 해야 하기 때문에, 가능하면 두 렌더 사이에 data와 treeNodes 데이터 값을 일정하게 유지하는 것이 좋음

## Usage - Selection

### Basic Selection

- `defaultSelectedItem`: 초기 selected item (uncontrolled 상태일 때만 적용 가능)

<Canvas of={TreeStories.BasicSelection} />

### Controlled Selection

- `selectedItem` prop을 통해 selection control 가능
- `selectedItem`: selected item
- `onSelectedItemChange`: selected item에 변화가 있을 때 호출되는 함수

<Canvas of={TreeStories.ControlledSelection} />

### Multi Selection

- 기본적으로 하나의 TreeItem만 선택 가능함 (Single Selection)
- `multiSelect`: true이면, 다중 selection이 가능함
- 특징
  - `Ctrl` 키를 누른 상태에서 items를 클릭하면 다수의 아이템들을 독립적으로 selection 가능함
  - `Shift` 키를 누른 상태에서 거리가 있는 두 items을 클릭하면 두 items를 포함한 사이의 모든 아이템들이 select 됨

<Canvas of={TreeStories.MultiSelection} />

### Checkbox Selection

### Automatic parents and children selection

#### 1. Propagate selection from parents to descendants

- 특징
  - 특정 한 parent를 select하면, 하위 모든 TreeItem들이 자동으로 select 됨
  - 특정 한 parent를 deselect하면, 하위 모든 TreeItem들이 자동으로 deselect 됨

<Canvas of={TreeStories.PropagateSelectionFromParentsToDescendants} />

#### 2. Propagate selection from descendants to parents

- 특징
  - 특정 한 parent의 하위 모든 TreeItem들을 select하면, parent가 자동으로 select 됨
  - select된 특정 한 parent의 하위 TreeItems 중 하나가 deselect되면, parent도 자동으로 deselect 됨

<Canvas of={TreeStories.PropagateSelectionFromDescendantsToParents} />
