# Carousel

각기 다른 content를 담은 slide 모음

## Import

```js
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselDots,
  CarouselFraction,
  CarouselPrevButton,
  CarouselNextButton,
  CarouselProgress
} from '@/components/data-display/Carousel';
```

## Usage

### Basic Carousel

- `Carousel` 컴포넌트

  - `children`: `CarouselContent`, `CarouselDots`, `CarouselFraction`, `CarouselProgress`, `CarouselPrevButton`, `CarouselNextButton` 등의 컴포넌트들

- `CarouselContent` 컴포넌트

  - `children`: `CarouselItem` 컴포넌트들

- `CarouselItem` 컴포넌트
  - `children`: 한 slide에 들어갈 contents

### Autoplay

- `Carousel` 컴포넌트
  - `autoplay`: true이면, 자동으로 slide가 넘어감
  - `autoplayDuration`: 한 slide당 유지 시간
  - `disableAutoplayOnInteraction`: true이면, carousel를 hover하거나 focus하거나 touch한 상태인 경우 autoplay 타이머가 잠시 중지됨
  - `onAutoplayLeftTimeChange`: 다음 slide로 넘어가기까지 남은 시간이 변경될 때 마다 호출되는 함수
