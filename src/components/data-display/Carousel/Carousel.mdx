# Carousel

각기 다른 content를 담은 slide 모음

- 구성 컴포넌트
  - `Carousel` 컴포넌트
  - `CarouselContent` 컴포넌트: carousel 내 모든 slides를 감싸는 wrapper
  - `CarouselItem` 컴포넌트: carousel 내 한 slide
  - `CarouselDots` 컴포넌트 (`Dots` 컴포넌트를 기반으로 함)
  - `CarouselDot` 컴포넌트 (`Dot` 컴포넌트를 기반으로 함)
  - `CarouselFraction` 컴포넌트 (`Fraction` 컴포넌트를 기반으로 함)
  - `CarouselProgress` 컴포넌트 (`LinearProgress` 컴포넌트를 기반으로 함)
  - `CarouselPrevButton` 컴포넌트 (`ButtonBase` 컴포넌트를 기반으로 함)
  - `CarouselNextButton` 컴포넌트 (`ButtonBase` 컴포넌트를 기반으로 함)

## Import

```js
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselDots,
  CarouselDot,
  CarouselFraction,
  CarouselPrevButton,
  CarouselNextButton,
  CarouselProgress
} from '@/components/data-display/Carousel';
```

### Snap free

#### 1. Snap free mode

- `snapMode`: snap 여부
  - 'snap': swipe가 끝났을 때, 임의의 기준점에 맞춰 자동으로 위치를 정렬
  - 'free': swipe 하는 만큼 이동함

<Canvas of={CarouselStories.SnapFreeMode} />

#### 2. Slip effect

- Slip Effect: swipe를 멈췄을 때, 멈출 때의 속도에 따라서 같은 방향으로 미끄러지는 효과
- free mode인 경우, 기본적으로 slip effect가 발생함
- `disableSlipEffect`: true이면, Slip Effect가 비활성화 됨
