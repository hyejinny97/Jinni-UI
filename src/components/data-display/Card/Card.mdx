# Card

카드

- 구성 컴포넌트
  - `Card` 컴포넌트: 하위 카드 관련 컴포넌트들을 감싸는 최상위 wrapper
  - `CardHeader` 컴포넌트: card header를 감싸는 wrapper
  - `CardBody` 컴포넌트: card content를 감싸는 wrapper
  - `CardFooter` 컴포넌트: card actions를 감싸는 wrapper

## Import

```js
import {
  Card,
  CardHeader,
  CardBody,
  CardFooter
} from '@/components/data-display/Card';
```
