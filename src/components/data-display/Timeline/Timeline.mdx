# Timeline

시간 순으로 일어난 이벤트 목록을 보여줌

- 구성 컴포넌트
  - `Timeline` 컴포넌트
  - `TimelineItem` 컴포넌트
  - `TimelineContent` 컴포넌트
  - `TimelineOppositeContent` 컴포넌트
  - `TimelineSeparator` 컴포넌트
  - `TimelineDot` 컴포넌트
  - `TimelineConnector` 컴포넌트

## Import

```js
import {
  Timeline,
  TimelineItem,
  TimelineContent,
  TimelineOppositeContent,
  TimelineSeparator,
  TimelineConnector,
  TimelineDot
} from '@/components/data-display/Timeline';
```
