# Timeline

시간 순으로 일어난 목록을 보여줌

## Import

```js
import {
  Timeline,
  TimelineItem,
  TimelineContent,
  TimelineOppositeContent,
  TimelineSeparator,
  TimelineConnector,
  TimelineDot
} from '@/components/data-display/Timeline';
```

## Usage

### Basic Timeline

- `Timeline` 컴포넌트

  - `children`: `TimelineItem` 컴포넌트들

- `TimelineItem` 컴포넌트

  - `children`: `TimelineContent`, `TimelineOppositeContent`, `TimelineSeparator` 컴포넌트들

- `TimelineContent` 컴포넌트

  - `children`: main content

- `TimelineSeparator` 컴포넌트

  - `children`: `TimelineDot`, `TimelineConnector` 컴포넌트

- `TimelineDot` 컴포넌트

  - `children`: dot 안에 들어갈 내용

- `TimelineConnector` 컴포넌트

<Canvas of={TimelineStories.BasicTimeline} />
