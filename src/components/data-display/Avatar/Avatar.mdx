# Avatar

사용자를 대표하기 위해 사용되는 컴포넌트로, 프로필 사진/닉네임/아이콘으로 나타낼 수 있다.

## Import

```js
import { Avatar } from '@/components/data-display/Avatar';
```

## Usage

### Image Avatars

- `src`: 이미지 url
- `alt`: 이미지 대체 텍스트
- `imgProps`: (src, alt 속성은 제외한) HTML img element의 props

<Canvas of={AvatarStories.ImageAvatars} />

### Letter Avatars

- `children`: letter or icon

<Canvas of={AvatarStories.LetterAvatars} />

### Icon Avatars

<Canvas of={AvatarStories.IconAvatars} />

### Sizes

- `size`: 아바타 크기
  - `xs` | `sm` | `md` | `lg` | `xl`: 차례대로 24px, 32px, 48px, 64px, 96px 크기
  - `number`: 가로/세로 모두 해당 설정값 크기

<Canvas of={AvatarStories.Sizes} />

### Shape

- `shape`: 아바타 형태
  - `circle`: 원 모양
  - `square`: 정사각형 모양
  - `rounded`: 모서리가 둥근 정사각형 모양

<Canvas of={AvatarStories.Shape} />
