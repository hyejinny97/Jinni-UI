# Tour

사용자에게 특정 위치마다 정보를 순서대로 제공함

## Import

```js
import { Tour, TourStep } from '@/components/data-display/Tour';
```

## Usage

### Basic Tour

- `children`: `TourStep` 컴포넌트들
- `value`: 현재 display 되는 `TourStep`의 value
- `open`: true이면, tour가 나타남
- `onClose`: `'escapeKeydown'`, `'backdropClick'` 이벤트 발생 시 호출되는 함수

<Canvas of={TourStories.BasicTour} />

### Mask props

- `Tour` 컴포넌트는 내부적으로 `Mask` 컴포넌트를 반환하므로, `Mask`의 일부 props 사용 가능
- `Tour`와 `TourStep` 컴포넌트에 동일한 prop을 적용한 경우, `TourStep` 컴포넌트에 적용한 prop 값이 우선 됨
