# Popover

화면 상단에 콘텐츠를 띄움

- 특징
  - Popover open 시, 스크롤 불가/backdrop 클릭 시 close 됨

## Import

```js
import { Popover } from '@/components/data-display/Popover';
```

## Usage

### Basic Popover

- `children`: popover의 콘텐츠
- `anchorEl`: anchor가 되는 HTML element로, popover의 위치를 결정해줌
- `open`: true이면, popover가 나타남
- `onClose`: `'escapeKeydown'`, `'backdropClick'` 이벤트 발생 시 호출되는 함수

<Canvas of={PopoverStories.BasicPopover} />

### Popover Position

- `popoverOrigin`: anchor의 origin에 부착할 popover의 origin
- `anchorReference`: popover의 위치를 설정할 때, 어떤 anchor prop을 참조할지 결정
  - 'anchorEl'(default) | 'anchorPosition'
- `anchorOrigin`: popover의 origin이 부착될 anchorEl의 origin
- `anchorPosition`: client area에서 메뉴의 상대적인 위치

### Elevation

- `PopoverContentProps`: PopoverContent(=Box)에 적용되는 props
  - elevation(default=5), round(default=4px), padding 등 변경 가능
