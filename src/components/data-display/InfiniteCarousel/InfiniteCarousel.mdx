# InfiniteCarousel

## Import

```js

  InfiniteCarousel,
  CarouselContent,
  CarouselItem,
  CarouselDots,
  CarouselDot,
  CarouselFraction,
  CarouselPrevButton,
  CarouselNextButton,
  CarouselProgress

```
