# Tooltip

사용자가 특정 요소에 hover, focus, tap 했을 때 정보성 텍스트를 보여줌

## Import

```js
import { Tooltip } from '@/components/data-display/Tooltip';
```

## Usage

### Basic Tooltip

- `children`: anchor 요소

  - 주의) anchor가 Component인 경우, 반드시 ref prop을 받을 수 있어야 함

- `content`: tooltip 콘텐츠

### Distance From Anchor

- `offset`: anchor와 tooltip 사이 거리

<Canvas of={TooltipStories.DistanceFromAnchor} />

### Triggers

- `triggers`: tooltip의 open을 유발하는 이벤트 종류

<Canvas of={TooltipStories.Triggers} />

### Controlled Tooltip

- `open`: true이면, tooltip이 나타남
- `onOpen`: triggers prop에 의한 open 이벤트 발생 시 실행되는 함수
  - open 이벤트: anchor click, mouse enter, focus
- `onClose`: triggers prop에 의한 close 이벤트 발생 시 실행되는 함수
  - close 이벤트: background click, mouse leave, blur

<Canvas of={TooltipStories.ControlledTooltip} />

### Showing and hiding delay

- `enterDelay`: tooltip이 나타날 때까지 delay 되는 시간 (단위: ms)
- `leaveDelay`: tooltip이 사라질 때까지 delay 되는 시간 (단위: ms)
