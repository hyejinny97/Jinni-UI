# AvatarGroup

아바타를 하나의 그룹으로 쌓은 컴포넌트

## Import

```js
import { AvatarGroup } from '@/components/data-display/AvatarGroup';
```

## Usage

### Max Avatar Number

- `max`: surplus avatar를 제외한 화면 상에 나타나는 최대 아바타 갯수

<Canvas of={AvatarGroupStories.MaxAvatarNumber} />

### Total Avatar Number

- `total`: 전체 아바타 갯수 (= 화면 상에 나타나는 아바타 갯수 + surplus 갯수)

<Canvas of={AvatarGroupStories.TotalAvatarNumber} />

### Custom Surplus Avatar

- `renderSurplus`: surplus avatar를 커스텀하는 함수
  - parameter: surplus (= surplus avatar 개수 = total - max)
  - return: `Avatar` 컴포넌트

<Canvas of={AvatarGroupStories.CustomSurplusAvatar} />

### Spacing

- `spacing`: 아바타 사이의 간격
  - `sm`: `Avatar` 너비 x 40% 만큼 겹쳐침
  - `md`: `Avatar` 너비 x 30% 만큼 겹쳐침
  - `lg`: `Avatar` 너비 x 20% 만큼 겹쳐침

<Canvas of={AvatarGroupStories.Spacing} />

### Sizes

- `size`: 하위 모든 `Avatar`의 크기를 한번에 변경해줌
  - `xs` | `sm` | `md` | `lg` | `xl`: 순서대로 24px, 32px, 48px, 64px, 96px 크기

<Canvas of={AvatarGroupStories.Sizes} />

### Shape

- `shape`: 하위 모든 `Avatar`의 모양을 한번에 변경해줌
  - `circle`: 원 모양
  - `square`: 정사각형 모양
  - `rounded`: 둥근 정사각형 모양

<Canvas of={AvatarGroupStories.Shape} />

### Color

HTML 요소 기본 `style` prop을 사용해 하위 모든 `Avatar` 색상 변경 가능

<Canvas of={AvatarGroupStories.Color} />
