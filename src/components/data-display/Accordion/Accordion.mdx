# Accordion

콘텐츠를 보여주거나(expand) 숨기는(collapse) 컴포넌트

- 구성 컴포넌트
  - `Accordion` 컴포넌트: 여러 개의 `AccordionItem`를 감싸는 wrapper
  - `AccordionItem` 컴포넌트: `AccordionSummary`, `AccordionDetails`를 감싸는 wrapper
  - `AccordionSummary` 컴포넌트: accordion header (trigger)
  - `AccordionDetails` 컴포넌트: accordion body (panel)

## Import

```js
import {
  Accordion,
  AccordionItem,
  AccordionSummary,
  AccordionDetails
} from '@/components/data-display/Accordion';
```

### Expanded by default

- `AccordionItem` 컴포넌트
  - `defaultExpanded`: true이면, 초기에 expand 됨 (uncontrolled인 경우에만 적용 가능)

<Canvas of={AccordionStories.ExpandedByDefault} />

### Controlled Accordion

- `expanded` prop을 통해 accordion을 control 할 수 있음
- `AccordionItem` 컴포넌트

  - `expanded`: true이면, expand 됨
  - `onChange`: expand/collapse state에 변화가 있을 때 호출되는 함수

<Canvas of={AccordionStories.ControlledAccordion} />

### Disabled Item

- `AccordionItem` 컴포넌트
  - `disabled`: true이면, 비활성화 됨

<Canvas of={AccordionStories.DisabledItem} />

### Expand Icon

- `AccordionSummary` 컴포넌트
  - `expandIcon`: expand indicator icon (`false`이면, expandIcon이 나타나지 않음)
