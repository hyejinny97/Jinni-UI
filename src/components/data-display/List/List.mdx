# List

연속적인 텍스트나 이미지 목록

## Import

```js
import { List } from '@/components/data-display/List';
```

## Usage

### Basic List

- `List` 컴포넌트

  - `children`: list items (ex) `ListItem`, `Divider` 컴포넌트 등)

- `ListItem` 컴포넌트
  - `children`: list item content
